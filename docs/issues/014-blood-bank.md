## What to build

Blood Bank management — donor management, inventory, and transfusion tracking.

1. **Donor Registration** — Donor demographics, contact, blood group, medical screening
2. **Blood Unit Inventory** — Track units by blood group, Rh factor, component (whole blood, RBC, plasma, platelets), collection date, expiry date
3. **Cross-matching** — Compatibility check between donor unit and recipient
4. **Blood Issue** — Issue blood to patient with cross-match confirmation, requesting doctor
5. **Transfusion Records** — Start time, end time, vitals monitoring, adverse reactions
6. **Donor Camp Management** — Organize camps, track collections
7. **Screening Test Results** — HIV, Hepatitis, Syphilis, etc. — mark unit as safe/unsafe

## Acceptance criteria

- [ ] Donor registered with medical screening results
- [ ] Blood units tracked by blood group and component with expiry dates
- [ ] Cross-match confirms compatibility before issue
- [ ] Expired/unsafe units automatically quarantined
- [ ] Blood group-wise stock level visible on dashboard
