## What to build

Public-facing content pages — about, departments, doctors, contact, careers.

1. **About Us** — Hospital mission, vision, leadership team, infrastructure, accreditations
2. **Departments & Services** — Department listing with icons, detail page with overview and services
3. **Doctor Directory** — Search/filter by department, specialization; doctor profile with qualifications, experience, OPD schedule
4. **Contact Us** — Contact form (name, email, phone, message → stored in DB + email to admin), Google Maps, phone/email directory
5. **Careers** — Job listings from admin, job detail, online application form, application confirmation
6. **Emergency Info** — ER contact, ambulance info, emergency preparedness guide
7. **Health Packages** — Package listing from admin, detail with inclusions/price, book online

## Acceptance criteria

- [ ] All pages render from CMS-managed content
- [ ] Doctor directory searchable and filterable
- [ ] Contact form submissions stored in database and emailed to admin
- [ ] Job applications stored and visible in admin panel
- [ ] Health package booking creates appointment in admin system

#3 (Public Layout), #5 (Doctor Management)
