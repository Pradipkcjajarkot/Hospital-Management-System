## What to build

Comprehensive dashboard and reporting module for actionable insights.

1. **Admin Dashboard** — KPI cards (bed occupancy %, today's appointments, revenue today, total patients), charts (monthly revenue trend, department-wise patients, appointment status distribution)
2. **Module-specific Dashboards** — Each module gets its own dashboard/widget (pharmacy stock alerts, lab pending tests, etc.)
3. **Custom Reports** — Filter by date range, department, report type; generate on-demand
4. **Pre-built Reports** — Patient demographics, revenue analysis, OPD/IPD statistics, pharmacy fast/slow moving, bed occupancy trends
5. **Export** — PDF, Excel, CSV export with formatting
6. **Scheduled Reports** — Email reports automatically on schedule (daily, weekly, monthly)
7. **Dashboard Widget Configuration** — Drag-and-drop widget arrangement, save layout per user

## Acceptance criteria

- [ ] Dashboard loads with real KPI data within 2 seconds
- [ ] Charts are interactive (hover, zoom, date range filter)
- [ ] Reports export to PDF/Excel with hospital logo and formatting
- [ ] Scheduled reports arrive via email on time
- [ ] Each user role sees role-relevant dashboard widgets
