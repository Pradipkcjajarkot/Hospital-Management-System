## What to build

Human Resources and Payroll management for hospital staff.

1. **Employee Master** — Staff records with personal info, department, designation, joining date, documents
2. **Attendance Tracking** — Daily attendance (present/absent/leave/holiday), biometric integration option
3. **Leave Management** — Leave types (sick, casual, annual), leave balance, application → approval workflow
4. **Payroll Processing** — Salary structure, deductions (tax, insurance, loan), generate payslip monthly
5. **Loan/Advance Management** — Employee loan/advance request, approval, repayment schedule
6. **Training Records** — Training programs attended, certificates, expiry tracking
7. **Employee Directory** — Searchable directory with department filter
8. **Reports** — Attendance summary, payroll summary, employee turnover

## Acceptance criteria

- [ ] Employee master with document upload and search
- [ ] Leave request → approval workflow works
- [ ] Monthly payroll generates payslips for all active employees
- [ ] Attendance report shows daily/monthly summary
- [ ] Loan repayment auto-deducts from salary
