## What to build

Public-facing website layout and home page, powered by CMS data.

1. **Base layout** — Header with logo + dynamic navigation menu, footer with contact/social links
2. **Home page** — Hero banner slider (from CMS content blocks), hospital stats counters, featured departments, latest news, testimonials
3. **Responsive design** — Mobile-friendly using Bootstrap/Tailwind
4. **Quick links** — Appointment booking, emergency info, contact
5. **Emergency notification bar** — Fixed banner with ER contact number

## Acceptance criteria

- [ ] Public home page loads with CMS-managed hero banners
- [ ] Navigation menu renders from CMS (pages, dropdowns)
- [ ] Footer displays hospital contact info, social links
- [ ] Fully responsive on mobile, tablet, desktop
- [ ] Emergency banner visible on all pages
